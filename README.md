# NL-bot-V3 (Frutnut)

NL-bot-V3 is a Discord bot built for the NL Oost community. It includes moderation tools, leveling, self-roles, economy features, and fun mini-games.

## Features
- Moderation commands
- Leveling system
- Self-roles
- Economy system
- Mini-games
- Slash commands
